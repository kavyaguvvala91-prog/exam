# Project Structure

This repository is organized so each part of the app has a clear home:

- `frontend/` for the React frontend
- `backend/` for the Node/Express backend
- `project/` for shared project docs and non-app resources

## Notes

- `render.yaml` stays at the repository root because Render expects to find it there for automatic deployment setup.
- `.gitignore` also stays at the root because it applies to the whole repository.
